默认值static : 不会被特殊定位,不会被"positioned"

相对定位relative : 设置 top 、 right 、 bottom 和 left 属性会使其偏离其正常位置,其他的元素的位置则不会受该元素的影响发生位置改变来弥补它偏离后剩下的空隙

绝对定位absolute : 相对于最近的"positioned"祖先元素定位,如果没有"positioned"祖先元素，那么它是相对于文档的 body 元素，并且它会随着页面滚动而移动

固定定位fixed : 会相对于视窗定位
